Definition 2: A relation R on a set A is called a universal relation if every element of A is related to every element of A; that is, R = A × A. [universal relation] concept=concept.cb2eaca4b418 field=definition_texts locator=0

1. Let \(A\) be the set of all students of a boys’ school.  
2. Define the relation \(R\) on \(A\) by  
   \[
   R=\{(a,b): a\text{ is sister of }b\}.
   \]  
   Because the school admits only boys, no student can be the sister of any other student; therefore \(R=\varnothing\), which is the empty relation.  
3. Define another relation \(R'\) on \(A\) by  
   \[
   R'=\{(a,b):\text{the difference between the heights of }a\text{ and }b\text{ is less than }3\text{ meters}\}.
   \]  
   It is clear that the height difference between any two students is always less than three meters, so \(R' = A \times A\), the universal relation.  
4. In class we have seen two ways to represent a relation: the raster (matrix) method and the set‑builder method. For example, on the set \(\{1,2,3,4\}\) the relation  
   \[
   R=\{(a,b): b = a + 1\}
   \]  
   can also be written as “\(a\,R\,b\) if and only if \(b = a + 1\)”, a notation many authors use when convenient.  
5. Whenever \((a,b) \in R\) we say that “\(a\) is related to \(b\)” and we denote this by \(a\,R\,b\).  
6. One of the most important kinds of relation in mathematics is an **equivalence relation**. To study equivalence relations we first examine three basic properties: reflexive, symmetric, and transitive.  
7. **Reflexive**: a relation \(R\) on a set \(A\) is reflexive if \((a,a) \in R\) for every \(a \in A\).  
8. **Symmetric**: a relation \(R\) on a set \(A\) is symmetric if \((a_1,a_2) \in R\) implies \((a_2,a_1) \in R\) for all \(a_1, a_2 \in A\).  
9. **Transitive**: a relation \(R\) on a set \(A\) is transitive if \((a_1,a_2) \in R\) and \((a_2,a_3) \in R\) together imply \((a_1,a_3) \in R\) for all \(a_1, a_2, a_3 \in A\).
