**Step 1.** Consider the claim \(x = e^{\log x}\).  
First note that the logarithm is defined only for positive real numbers, so the equation cannot hold for non‑positive \(x\).  
Set \(y = e^{\log x}\). If \(y>0\) we may take logarithms of both sides, giving \(\log y = \log (e^{\log x}) = \log x\). Hence \(y = x\).  
Therefore the identity \(x = e^{\log x}\) is valid only when \(x\) is positive.  

**Step 2.** Recall two fundamental differentiation formulas (proved elsewhere):  
1. \(\displaystyle \frac{d}{dx}\,e^{x}=e^{x}\).  
2. \(\displaystyle \frac{d}{dx}\,\log x=\frac{1}{x}\).  

**Step 3.** Differentiate the following functions with respect to \(x\) using the chain rule.  

&nbsp;&nbsp;(i) \(y = e^{-x}\).  
 \(\displaystyle \frac{dy}{dx}= -\,e^{-x}\).  

&nbsp;&nbsp;(ii) \(y = \sin(\log x)\) (with \(x>0\)).  
 \(\displaystyle \frac{dy}{dx}= \cos(\log x)\cdot\frac{1}{x}\).  

&nbsp;&nbsp;(iii) \(y = \cos^{-1}(e^{x})\).  
 \(\displaystyle \frac{dy}{dx}= -\frac{e^{x}}{\sqrt{1-e^{2x}}}\).  

&nbsp;&nbsp;(iv) \(y = e^{\cos x}\).  
 \(\displaystyle \frac{dy}{dx}= -\,e^{\cos x}\,\sin x\).  

**Step 4.** Differentiate  
\[
y=\frac{\sqrt{x-3}\,\sqrt{x^{2}+4}}{\sqrt{3x^{2}+4x+5}} .
\]  
Take natural logarithms:  
\[
\log y=\tfrac12\bigl[\log(x-3)+\log(x^{2}+4)-\log(3x^{2}+4x+5)\bigr].
\]  
Differentiate both sides and solve for \(\displaystyle \frac{dy}{dx}\).  

**Step 5.** Differentiate the exponential function with a constant base \(a>0\):  
\(y = a^{x}\).  
Write \(\log y = x\log a\). Differentiating gives \(\displaystyle \frac{1}{y}\frac{dy}{dx}= \log a\), so  
\[
\frac{dy}{dx}=a^{x}\log a .
\]  

**Step 6.** Differentiate \(y = x\sin x\) for \(x>0\).  
Take logarithms: \(\log y = \log x + \log(\sin x)\).  
Differentiating yields  
\[
\frac{1}{y}\frac{dy}{dx}= \frac{1}{x} + \cot x .
\]  
Hence \(\displaystyle \frac{dy}{dx}= y\!\left(\frac{1}{x}+\cot x\right)= x\sin x\!\left(\frac{1}{x}+\cot x\right).\)  

**Step 7.** Find \(\displaystyle \frac{dy}{dx}\) from the relation  
\[
y^{x}+x^{y}+x = a^{b}.
\]  
Introduce \(u = y^{x},\; v = x^{y},\; w = x\).  
Differentiate each term using logarithmic differentiation, then combine the results to solve for \(\displaystyle \frac{dy}{dx}\).
